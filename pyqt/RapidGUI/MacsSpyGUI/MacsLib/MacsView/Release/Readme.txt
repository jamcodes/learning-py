Usage: MacsView port MacsMsg.ini [MacsView.ini]

Options:
 port          0 = A.M.I.
 MacsMsg.ini   Macs messages configuration file
 MacsView.ini  Macs viewer configuration file

Keys:
 Z             exit
 z             select service (if available)
 H             reload MacsView.ini (if available)
 h             service help (if available)
